Salin hasilnya ke `ADMIN_PASSWORD_HASH` di Render.

| Key | Keterangan |
|-----|-----------|
| `SESSION_SECRET` | String acak ≥32 karakter |
| `JWT_SECRET` | String acak ≥32 karakter |
| `ADMIN_PASSWORD_HASH` | Hash bcrypt dari password admin |
| `CLOUDINARY_CLOUD_NAME` | Dari dashboard Cloudinary |
| `CLOUDINARY_API_KEY` | Dari dashboard Cloudinary |
| `CLOUDINARY_API_SECRET` | Dari dashboard Cloudinary |
| `PROVIDER_API_KEYS` | JSON array key per provider |
